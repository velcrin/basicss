## Border

e.g. `border around` or `border top bottom strong-sand dashed`

`border` + position + [ color x weight x style ] 

 + weight `thick` | `thin`
 + position `around` | `top` x `right` x `bottom` x `left`
 + style `dotted` | `dashed` | `solid` | `double` | `groove` | `ridge` | `inset` | `outset`

## Margin
